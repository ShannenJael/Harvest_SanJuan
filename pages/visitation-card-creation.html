Visitation Card Creation

Create printable visitation cards from a spreadsheet

Login Required

Use the same credentials as the Visitation Report page.

Invalid credentials.

Upload Spreadsheet

Upload an Excel (.xlsx, .xls) or CSV file to generate printable visitation cards. Each page will contain 2 cards.

Spreadsheet Format:

Your spreadsheet should have columns with these headers (case-insensitive):

Name or personName - Person's name (required)
Address - Street address (required)
Distance - Distance from church (optional)

Example spreadsheet: Name | Address | Distance ----------------|------------------------------|---------- Samantha Remson | 135 NEWCASTLE CIR | 0.6 miles Donald White | 920 DENTON BLVD NW APT 261 | 0.7 miles

Tip: The generated PDF will have 2 cards per page, pre-filled with name, address, and distance. All other fields are left blank for handwriting after the visit.
